# Task Management Board

### DUMP Internship - JavaScript #3 (Node.js)

## Task

Build a **Task Management Board** inspired by **Trello / GitHub Projects** using **vanilla JavaScript** on the frontend and **Node.js** with **PostgreSQL** on the backend.

## Features

- Board with task columns (Blocked, Todo, In Progress, In Review, Done)
- Create, edit, move (drag & drop), archive and delete tasks
- Task details shown in popup or sidebar
- Archived tasks in a separate view
- Visual indicators for upcoming deadlines
- Dark / Light mode

## Bonus

- User authentication with hashed passwords
- Protected app access (JWT or cookies)

## Tech Stack

- HTML, CSS, JavaScript
- Node.js, Express
- PostgreSQL (`pg`)

## Setup

1. Clone the repository
2. Install dependencies
3. Run backend and open frontend
